## [1.0.0] - 2026-01-13
### Added
- Initial public release
- Safe MIUI debloat (disable-only, no uninstall)
- Clean system (kill background + trim cache)
- Game Mode ON / OFF
- Ultra Battery Mode ON / OFF
- RAM & CPU monitor (top)
- Auto-detect Redmi / Xiaomi / POCO devices
- ASCII logo branding (myrul.dev)

### Notes
- Tested on Redmi Note 7 (MIUI, Android 10+)
- Requires Termux + Shizuku (no root)
